Add data-toggle="collapse" and a data-target to an element to automatically assign control of a collapsible element. The data-target attribute accepts a css selector to apply the collapse to. Be sure to add the class collapse to the collapsible element. If you'd like it to default open, add the additional class in.

To add accordion-like group management to a collapsible control, add the data attribute data-parent="#selector".

NOTE: See Bootstrap documentation for a full list of options, methods and events for controlling collapsibles.

Add .form-horizontal to align all controls and labels horizontally.

      <label class="control-label">Text input</label>
      <div class="controls">

      </div>

Buttons

Example Description Usage

Secondary

Default
Classes Usage
btn btn-primary Primary action button.
btn btn-link Secondary button used for Read More and similar navigation

Use the .btn class on an <a>, <button>, or <input> element.

<a class="btn" href="">Link</a>
<button class="btn" type="submit">Button</button>
<input class="btn" type="button" value="Input">
<input class="btn" type="submit" value="Submit">

Buttons should only be used for actions while hyperlinks are to be used for objects. For instance, "Download" should be a button while "recent activity" should be a link.

As a best practice, try to match the element in context to ensure matching cross-browser rendering. If you have an input, use an <input type="submit"> for your button.

Multiple Sizes

Add .btn-large, .btn-small, or .btn-mini for additional sizes.

<button class="btn btn-large">Button</button>

Disabled

For disabled buttons, add the .disabled class to links and the disabled attribute for <button> elements.

<a class="btn disabled" href="">Link</a>
<button class="btn disabled" disabled="disabled">Button</button>

Button Groups

Use button groups to join multiple buttons together as one composite component. Build them with a series of <a> or <button> elements.

<div class="btn-group">
  <button class="btn">Left</button>
  <button class="btn">Middle</button>
  <button class="btn">Right</button>
</div>
Button Toolbar

Combine sets of <div class="btn-group"> into a <div class="btn-toolbar"> for more complex components.

<div class="btn-toolbar">
  <div class="btn-group">
    …
  </div>
</div>

When using beside strings of text, as in buttons or nav links, be sure to leave a space after the <i> tag for proper spacing.
